# Netstat Monitor

## Описание
Java-приложение для мониторинга сетевых соединений. Программа использует системную утилиту `netstat`, анализирует её вывод и предоставляет отчет о:
1. Процессах, слушающих порты (LISTEN).
2. Внешних соединениях (ESTABLISHED).
3. Статусе конкретного порта (занят/свободен).

Приложение работает корректно как на **Windows**, так и на **Linux** (автоопределение ОС).

**Технологии:** Java 21, Gradle, JUnit 5.

## Сборка проекта
Проект использует систему автоматической сборки Gradle.
Для создания исполняемого JAR-файла выполните команду в терминале:

```bash
# Windows
gradlew build

# Linux / MacOS
./gradlew build
