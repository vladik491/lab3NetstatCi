## 2. О проекте
**Netstat Monitor** — это приложение, которое показывает сетевую активность компьютера в понятном виде. Оно работает как "переводчик" для системной команды `netstat`.

**Возможности:**
1. Показывает программы, которые "слушают" порты.
2. Показывает, с какими сайтами соединен компьютер.
3. Проверяет, свободен ли конкретный порт (например, 8080).

## 3. Подготовка (Установка Java)
Для работы программы обязательна **Java версии 21**.

### Шаг 1. Проверка
1. Нажмите `Win` + `R`, введите `cmd`, нажмите **Enter**.
2. Напишите: `java -version`
    - Если написано `openjdk version "21..."` — **всё отлично**.
    - Если ошибка или версия ниже 21 — нужна установка.

### Шаг 2. Установка (если нужно)
1. Скачайте JDK 21 (x64 Installer) с официального сайта:
   https://www.oracle.com/java/technologies/downloads/#java21
2. Установите его, нажимая Next.
3. **Перезагрузите компьютер**.

## 4. Настройка (Создание файла app.properties)
**Внимание!** Программа не запустится без этого файла. Вы должны создать его сами в той же папке, где лежит файл `.jar`.

1. Зайдите в папку с программой.
2. Нажмите правую кнопку мыши -> **Создать** -> **Текстовый документ**.
3. Переименуйте файл в `app.properties`.

   !!! **ОЧЕНЬ ВАЖНО ПРО РАСШИРЕНИЕ:**
   Windows часто скрывает расширения файлов (буквы после точки).
   Если вы назвали файл `app.properties`, а он остался текстовым, на самом деле он называется `app.properties.txt`. **Программа его не увидит!**
   *Как проверить:* Нажмите в папке сверху "Вид" -> поставьте галочку "Расширения имен файлов". Имя файла должно заканчиваться на `.properties`, а не на `.txt`.

4. Откройте файл Блокнотом и вставьте текст:

```properties
# Порт для проверки (Задание №3)
monitor.port.check=8080

# Режим отчета: VERBOSE (подробно) или SUMMARY (кратко)
output.mode=VERBOSE
```
5. Сохраните файл.

## 5. Запуск приложения
Теперь в папке два файла: `.jar` (программа) и `.properties` (настройки).

1. Кликните мышкой по **адресной строке** папки (сверху).
2. Напишите `cmd` и нажмите **Enter**. Откроется черное окно.
3. Введите команду:
   ```bash
   java -jar NetstatMonitor-1.0-SNAPSHOT.jar
   ```
4. Нажмите **Enter**.

**Пример результата:**
```text
Mode: VERBOSE

--- Listening Ports ---
PID: 1234, Process: mysqld, Port: 3306

--- External Connections ---
PID: 5678, Process: chrome, Foreign: 104.18.3.10:443

--- Port 8080 Check ---
Port 8080 is not in use.
```
